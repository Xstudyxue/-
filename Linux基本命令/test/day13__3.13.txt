打包压缩
1. 使用tar打包/var/log/目录，压缩包的名字为log.tar.gz,压缩包放在 /opt 目录下
# tar -czf /opt/log.tar.gz /var/log
2. 使用zip打包/etc目录。
# zip  /opt etc.zip /etc
3.查看log.tar.gz压缩包中有哪些内容。
# tar -tf /opt/log.tar.gz
4. 用zip打包/opt目录，要求不显示打包过程
# zip -q opt.zip /opt
5. 找出系统内所有 .log 结尾的普通文件，并进行压缩打包，压缩包放在 /tmp 下，压缩包的名字叫 log.tar.xz
# find / -name "*.log" -type f | xargs -I {} tar -cjf /tmp/log.tar.xz {}
